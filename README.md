# Agent Web GUI

This project provides a web-based Graphical User Interface (GUI) for interacting with a local Ollama LLM agent. The agent is capable of using tools (function calling) to answer user queries.

## Features

*   **Real-time Streaming:** Responses from the LLM are streamed to the UI in real-time.
*   **Tool Execution:** The agent can autonomously decide to call Python functions (e.g., `get_weather`, `recommend_activity`) and use their results.
*   **Conversation Memory:** The agent remembers previous messages within the same session.
*   **Markdown Support:** The chat interface fully supports Markdown formatting.
*   **Syntax Highlighting:** Code blocks generated by the agent are automatically colorized using `highlight.js`.
*   **Mathematical Formulas:** Supports rendering of LaTeX math formulas using `KaTeX`.
*   **Auto-growing Input:** The chat input box automatically expands as you type multi-line prompts.

## Prerequisites

Before running the server, ensure you have the following installed:

1.  **Python 3.8+**
2.  **Ollama:** You must have Ollama installed and running locally.
    *   Download from: [https://ollama.com/](https://ollama.com/)
    *   Ensure you have pulled the required model. By default, this project uses `qwen3:1.7b`.
    *   Run this command in your terminal to download the model:
        ```bash
        ollama run qwen3:1.7b
        ```

## Installation

1.  Clone or download this repository.
2.  Navigate to the project directory in your terminal.
3.  Install the required Python dependencies using `pip`:

    ```bash
    pip install fastapi uvicorn ollama docstring_parser
    ```

## Running the Server

1.  Ensure your local Ollama application is running in the background.
2.  Start the FastAPI web server by running the following command in your terminal:

    ```bash
    python web_app.py
    ```

3.  You should see output indicating the server has started:
    ```
    Starting web server at http://localhost:8000
    INFO:     Started server process [XXXXX]
    INFO:     Waiting for application startup.
    INFO:     Application startup complete.
    INFO:     Uvicorn running on http://0.0.0.0:8000 (Press CTRL+C to quit)
    ```

## Usage

1.  Open your web browser and navigate to: **http://localhost:8000**
2.  Type your prompt in the input box at the bottom.
    *   Press `Enter` to send the message.
    *   Press `Shift + Enter` to add a new line to your prompt.
3.  Watch as the agent thinks, executes tools (if necessary), and streams the final response back to you!

## Project Structure

*   `web_app.py`: The main Python backend using FastAPI. It defines the tools, handles communication with the Ollama API, and serves the web endpoints.
*   `static/index.html`: The frontend HTML, CSS, and JavaScript that provides the chat interface.
*   `main.py`: A CLI version of the agent (for terminal use only).
